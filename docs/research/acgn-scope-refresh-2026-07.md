# 1995—2005 馆藏 ACGN 范围刷新

核验日期：2026-07-22  
核验对象：`src/data.js` 当前 95 条馆藏，重点为 17 条“音乐”和 22 条“小说”  
实施目标：任何条目都必须能直接归入 Animation / Comics / Games / Novels，不能再以泛流行文化或“年代代表作”为收录理由。

## 结论

当前动画、漫画、游戏条目按媒介身份天然属于 A/C/G；图像小说属于 Comics，欧美动画同样属于 Animation，不要求作品必须产自日本。真正越界的是：

- 17 条音乐里，除 `Discovery` 与动画电影《Interstella 5555》存在直接企划关系外，其余都是独立流行音乐。为避免边界争议，本方案仍建议 17 条全部换成同年动画/游戏的官方原声或企划音乐。
- 22 条小说虽然在字面上都属于 Novel，但其中 16 条是普通严肃文学、现实主义或与 A/C/G 无明确联系的文学小说。严格馆藏口径下只保留 6 条，替换 16 条；替换后仍保持 1995—2005 每年 2 条小说。
- 视觉小说在产业分类上通常首先是 Games。若以后引入，应放入“游戏”频道并以“视觉小说”作格式或标签，不建议仅因中文名含“小说”就放入“小说”频道。

## 建议采用的范围规则

### 必收范围

1. Animation：电视动画、动画电影、OVA、网络动画，不限地区。
2. Comics：漫画、comics、bandes dessinées、manhwa、graphic novel。
3. Games：电子游戏、桌面角色扮演游戏、卡牌游戏；视觉小说归在这里。
4. Novels：轻小说；科幻、奇幻、恐怖、反乌托邦、冒险等类型小说；或与动画、漫画、游戏存在直接改编、衍生、共同世界、企划关系的小说。
5. Music：不作为独立的第五种文化门类。若 UI 继续保留“音乐”频道，条目本身必须是 A/C/G 作品的官方原声、角色歌、主题曲合集，或从立项时就与动画/漫画/游戏共同构成的跨媒体企划。

### 排除范围

- 仅因“很有时代代表性”而收录的普通摇滚、流行、嘻哈、爵士专辑。
- 普通严肃文学、现实主义社会小说、家族小说、历史文学，即使后来有真人影视改编，也不能只靠真人电影/电视剧关系进入 ACGN。
- 仅仅“谈论漫画/游戏”的文学作品；主题相关不等于作品自身属于 ACGN。
- 虚拟艺人或带动画 MV 的普通专辑，除非动画从一开始就是作品叙事与发行结构的一部分。

### 音乐替换的实施注意

- 页面上的类型可继续显示“音乐”，但英文元数据建议从宽泛的 `MUSIC` 改为 `ACGN SOUNDTRACKS`，或至少把 `format` 统一成 `ANIME OST`、`GAME OST`、`MEDIA-MIX ALBUM`。
- 每条说明第一句必须直接写明它服务于哪部动画/游戏；不要只描述曲风。
- 原编号可以保留，旧封面和旧 Wikimedia 来源记录必须一并替换，不能出现“数据已换、图片仍是旧专辑”的错配。
- 若希望保留 `Discovery`，最低限度应把条目名称改成 `Discovery / Interstella 5555` 跨媒体企划，并在来源中证明动画与专辑的共同构成关系；本报告仍推荐用 `Final Fantasy X Original Soundtrack` 取代，边界最清楚。

## 其余馆藏的范围审查

- 动画频道：现有条目均是电视动画、动画电影或 OVA；`South Park` 等欧美成人动画仍属于 Animation，应保留。
- 漫画频道：现有日漫、韩漫、美漫、法语漫画与 graphic novel 均属于 Comics；`Blankets`、`Jimmy Corrigan` 等无需因“文学性强”而删除。
- 游戏频道：现有条目均是电子游戏；属于 Games，应保留。
- 结论只针对媒介范围，不代表所有现有年份、封面权利、中文名和作品说明都已完成二次校对。

## 落库顺序

1. 先替换 17 条音乐，确保页面不再出现任何独立流行专辑。
2. 再按编号替换 16 条小说；6 条保留项只需调整标签，让“奇幻／科幻／跨媒体”归属可见。
3. 同步更新 Wikimedia 查询脚本中的标题、QID、固定 revision id、来源文档与封面清单。
4. 跑数据测试时新增两条范围断言：音乐条目必须包含 `Anime OST` / `Game OST` / `Media Mix` 标签之一；小说条目必须包含轻小说、科幻、奇幻、恐怖、反乌托邦、冒险或 A/C/G 改编关系之一。
5. 最后逐年检查数量：1995—2005 每年仍为 2 条小说；馆藏总数保持 95，不以删减代替更新。
